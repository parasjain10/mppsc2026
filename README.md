# MPPSC 2026 Smart Question Bank

AI-curated high-probability questions for MPPSC Prelims 2026 (Paper I - General Studies).

**Live Site:** [https://parasjain10.github.io/mppsc2026/](https://parasjain10.github.io/mppsc2026/)

## Features

- **136 high-probability questions** across 10 categories
- **Study Mode** — see all answers and detailed explanations
- **Quiz Mode** — test yourself with real exam-style negative marking (+2/-0.66)
- **Category filters** — MP GK, Current Affairs, Polity, History, Geography, Science, Economy, ICT, Sports
- **Probability filters** — focus on High or Medium probability questions
- **Search** — find questions by keyword
- **Progress tracker** — track how many questions you've attempted

## Exam Details

- **Date:** 26 April 2026
- **Paper I:** 100 Questions, 200 Marks (Merit-based)
- **Paper II:** CSAT (Qualifying)
- **Negative Marking:** 1/3 of marks for wrong answer
